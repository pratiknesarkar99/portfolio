# Personal Portfolio Website

A modern, responsive portfolio website built with React and React Bootstrap.

**Live Demo:** [https://pratiknesarkar99.github.io/portfolio](https://pratiknesarkar99.github.io/portfolio)

## Tech Stack

- React 18
- React Bootstrap 5
- React Router v6
- EmailJS (contact form)
- animate.css

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Runs the app at [http://localhost:3000](http://localhost:3000)

### Build

```bash
npm run build
```

### Deploy

```bash
npm run deploy
```

Deploys to GitHub Pages.

## Features

- Responsive design
- Smooth scroll navigation
- Animated sections
- Contact form with email integration
- Experience and project showcases
